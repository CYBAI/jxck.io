各バリデータへの態度

特に標準フォーマットについては、バリデータなどが提供されており、このサイトもそうしたツールを利用してテストを行っている。

本来であれば、そのバリデータにおいてオールグリーンとなるような修正をするところであるが、本サイトでは技術的な理由によって一部のバリデータのエラーを意図的に無視している

ここでは、各バリデータとそこで無視しているエラーについて、その理由を記す

W3C HTML Validator

https://validator.w3.org/ において HTML マークアップの妥当性について検証が可能である。

本サイトでは以下のエラーを無視している。

body 内での CSS 読み込みによるエラー
本サイトが実施している最適化のため、 body 内で意図的に CSS への >link< を入れている。

W3C CSS Validator

https://jigsaw.w3.org/css-validator/ において CSS の妥当性について検証が可能である。

本サイトでは以下のエラーを無視している。

pointer-events 利用によるエラー
本サイトでは、疑似要素のみリンク可能にするスタイルを採用しており、これを実現するために必要なため無視している。

本サイトでは以下のエラーを無視している。
